Why I Run a Home Server on an Old Laptop

Most people throw away their old laptops. I turned mine into a Debian home server — CasaOS for management, Immich for photo backups, Tailscale so I can access everything from anywhere. It hosts my WhatsApp bot too. Total cost: zero. Sometimes the best server is the one you already own.
